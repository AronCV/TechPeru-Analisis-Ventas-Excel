# 📊 TechPeru — Análisis de Ventas Excel 2024

Dashboard profesional de análisis de ventas construido en Excel, 
simulando el entorno de una empresa tecnológica peruana (TechPeru S.A.C.) 
con 279 registros de ventas durante el año 2024.

## 🗂️ Estructura del Proyecto

| Hoja | Descripción |
|---|---|
| KPI Dashboard | Indicadores generales: ingresos, ganancias, mejor vendedor y categoría TOP |
| Análisis | Consultas dinámicas por vendedor, categoría y región + búsqueda por ID |
| Reporte Mensual | Ventas mes a mes con gráfico de tendencia |
| Ranking | Top vendedores y categorías con % de participación y gráficos |
| Tabla Dinámica | Análisis interactivo con slicers por categoría y región |
| Ventas | Base de datos con 279 registros (datos en bruto) |
| Empleados | Directorio de 15 empleados con salarios y fechas de ingreso |

## 🧮 Fórmulas Utilizadas

BUSCARV, BUSCARX, INDICE+COINCIDIR, SI, SI+Y, SI+O,
SUMAR.SI, SUMAR.SI.CONJUNTO, CONTAR.SI, CONTAR.SI.CONJUNTO,
PROMEDIO.SI, PROMEDIO.SI.CONJUNTO, SUMAPRODUCTO,
IZQUIERDA, DERECHA, EXTRAE, CONCATENAR,
HOY, AÑO, MES, FIN.MES, SIFECHA,
MAX, MIN, MEDIANA, Formato Condicional

## 🛠️ Herramientas
- Microsoft Excel 365
- Tablas Dinámicas + Slicers
- Gráficos: línea, barras y pastel
